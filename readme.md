## What`s DyWeb?
* 从未间断的创新。我们不满足现状，我们渴求知识；我们钟情实践，希望通过代码改变我们的互联网。这里是我们释放自己能量的地方，我们致力于革新交大网络水平。
* 学习与交流的胜地。Linux, and Git
* 强大而又有爱的团队。这里有可爱的研究生学长顾问，有代码经验丰富的大四狗；有细心教导新人的阳光学长，也有负责卖萌的吉祥物学姐；有一心扑在自己项目上的梦想家，也有孜孜不倦对网络世界充满好奇的探索者……
* 我们在这里相聚，道同者相谋。
